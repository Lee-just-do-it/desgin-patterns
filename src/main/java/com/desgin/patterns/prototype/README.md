原型模式
   定义：针对一个类采用克隆的方式创建一个副本
优点：
   在内存中二进制流拷贝，比new性能要好，特别是在一个循环体中产生大量对象时，可以考虑原型模式

使用场景：
    一个对象需要提供给其他对象访问，而且各个调用者 可能需要修改其值      
    
其实就是：
    class A implements Cloneable{
        
        重写clone方法
        注意：引用层级关系是否需要深拷贝 还是浅拷贝
             层级引用太深，是否需要考虑流拷贝等情况
    }      
    
    通过  A a = ainstance.clone()
