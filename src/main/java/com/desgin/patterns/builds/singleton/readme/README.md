单例模式
定义：确保一个类自行实力划只能产生一个实例，并且向整个系统提供这个实例

单例的实现有很多种，反射等，但是类本身有个构造器，这里用构造器的方式

几种常用的单例模式：
     饿汉模式
     懒汉模式
     双重检测锁模式 
     静态内部类模式（推荐）
